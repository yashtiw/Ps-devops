# Lambda + API Gateway CDK App

This project creates an AWS Lambda function fronted by an HTTP API Gateway using AWS CDK (TypeScript).

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Bootstrap your AWS environment if needed:
   ```bash
   npx cdk bootstrap
   ```

3. Deploy the stack:
   ```bash
   npm run deploy
   ```

## GitHub

1. Initialize git locally:
   ```bash
   git init
   git add .
   git commit -m "Initial CDK Lambda + API Gateway scaffold"
   ```

2. Add your GitHub remote and push:
   ```bash
   git remote add origin https://github.com/<your-user>/<your-repo>.git
   git branch -M main
   git push -u origin main
   ```
